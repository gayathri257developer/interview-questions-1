5. Difference between the block and inline?

A block element is always begin in a new line and it will take full width of the webpage or screen. by default browser add some space before and after the element.
Examples : h1 to h6, p, div, table, form etc.

An inline element does not begin on a new line, it takes only the space that is required for that particual text or content.
Examples :- span, a tag, img, input etc.
